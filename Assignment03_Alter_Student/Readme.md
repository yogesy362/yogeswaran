# Assignment 3 – Alter Student Table

## Objective

Modify the existing Student table.

## Problem Statement

Alter the Student table by adding the following columns:

| Column Name | Data Type |
|-------------|-----------|
| Email | VARCHAR(30) |
| PhoneNumber | NUMBER(10) |

After modifying the table, display its structure using the `DESCRIBE` command.

## Instructions

- Write your SQL statements in `solution.sql`.
- Use the ALTER TABLE statement.
- Display the modified table structure.

## Expected Outcome

The Student table should contain the additional columns **Email** and **PhoneNumber**.
