# Assignment 4 – Create Course Table

## Objective

Create a Course table and insert sample records.

## Problem Statement

Create a table named **Course** with the following fields:

| Column Name | Data Type |
|-------------|-----------|
| CourseID | NUMBER(5) |
| CourseName | VARCHAR(30) |
| Credits | NUMBER(2) |
| DepartmentID | NUMBER(5) |

Insert at least **three records** into the table.

Display the table structure using the `DESCRIBE` command.

## Instructions

- Write your SQL statements in `solution.sql`.
- Insert a minimum of three records.
- Display the table structure.

## Expected Outcome

The Course table should be created successfully with at least three records.
