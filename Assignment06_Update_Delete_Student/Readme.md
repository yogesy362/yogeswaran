# Assignment 6 – Update and Delete Student Records

## Objective

Perform UPDATE and DELETE operations on the Student table.

## Problem Statement

1. Update the DepartmentID of student **Karthik** from **101** to **103**.
2. Delete the student whose **StudentID = 1002**.
3. Display the updated Student table.

## Instructions

- Write all SQL statements in `solution.sql`.
- Use UPDATE and DELETE statements.
- Display the final contents of the Student table.

## Expected Outcome

- Karthik should belong to Department 103.
- StudentID 1002 should no longer exist.
